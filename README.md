# homebrew-tap

Homebrew tap for [figma-cli](https://github.com/AndVl1/figma-cli).

## Install

```bash
brew tap AndVl1/tap
brew install figma-cli
```

## Tools

| Formula | Description |
|---------|-------------|
| `figma-cli` | Rust CLI for Figma API — token-efficient alternative to figma-console-mcp |
